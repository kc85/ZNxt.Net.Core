Overview

ZNxt.App is an open source, cross platform, highly scalable, micro-services based application development framework. ZNxt.App designed on .net core 2.2 and can run on standalone on under container. Its has build it support of authentication, authorization, micro front end, dynamic module injection.

Docker commands


#Basic setup
sudo docker run -d -p 806:80 -p 4436:443 --restart=always --name znxt-app-run choudhurykhanin/znxtapp:latest

#Mapped static content on windows ## c:\myapp\wwwroot is the folder from your application static content
sudo docker run -d -p 806:80 -p 4436:443 -v c:\myapp\wwwroot:/app/wwwroot --restart=always --name znxt-app-run choudhurykhanin/znxtapp:latest

#Mapped static content on linux ## /myapp/wwwroot is the folder from your application static content
sudo docker run -d -p 806:80 -p 4436:443 -v /myapp/wwwroot:/app/wwwroot --restart=always --name znxt-app-run choudhurykhanin/znxtapp:latest

Create Api Module

Prerequisite: .Net Core 2.2
 
  • Install ZNxt Module templete
    #Run below command for your command prompt or terminal
    dotnet new --install ZNxt.Module.Template.CSharp::1.0.0
     
  • Create new C# module project
    # Create folder with you module name. (eg _> md MyModule.EComm). From cmd/terminal go inside the folder and run below command
    dotnet new znxtmodule --ModuleName EComm --Namespace MyModule --ApiPrefix ecom
     
  • Build Project
    # Go to the folder yournamespace.yourmodulename.Web.Test
    eg. _> cd MyModule.EComm.Web.Test
    # Do dotnet build
    _> dotnet build
     
  • Run The Project
    _> dotnet run
     
  • Open below link to you browser
    _> http://localhost:880
     
  • Call Sample API
    http://localhost:880/api/{yourapiprefix}/ping
    e.g: http://localhost:880/api/EComm/ping
